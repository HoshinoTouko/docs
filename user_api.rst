API 文档
==========================

Api 地址说明
---------------------------

http://HOST:PORT/api/{VERSION}/{APP}/{RESOURCE}/
http://HOST:PORT/api/{VERSION}/{APP}/{RESOURCE}/{RESOURCEID}/

如:

- 资产列表

    http://localhost/api/v1/assets/assets/

- 资产详情

    http://localhost/api/v1/assets/assets/2c56fd37-db65-40ed-b787-b65a98635f12/
